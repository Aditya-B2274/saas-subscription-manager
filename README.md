SaaSify is a SaaS Subscription Management Platform.

Tech Stack: 

Next.js, Node.js, PostgreSQL, Stripe API.

Features :

Subscription plans & upgrades ,

Automated billing & invoices ,

Usage analytics dashboard ,

Role-based admin panel.
